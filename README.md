## Installation

To get a local copy up and running, follow these steps:

1. **Clone the repository:**

   ```bash

2. **Navigate to the project directory:**

   ```bash
   cd your-repo-name

3. Install the dependencies:

   ```bash
   npm install

3. Start the development server:

   ```bash
